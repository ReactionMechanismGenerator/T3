tandem.logger¶

t3 logger module

Using a custom logger to avoid intervening with RMG’s / ARC’s loggers.

tandem.logger.log(message: str, level: Optional[str] = 'info', verbose: Optional[bool] = True) → None[source]¶

The tandem tool logging function. RMG and ARC have loggers that will override a conventional logger used here imported from logging Hence we define our own simple logging tool here.

Parameters

  • message (str) – The message to be logged.

  • level (str, optional) – The message level. Controls the prefix and suffix to be added to the message. Allowed values are: ‘info’ (default), ‘warning’, and ‘error’.

  • verbose (Optional[bool]) – Whether or not to log to file.

Report species summary. The report will be saved as RMG_ARC_species.log under the run_directory the RMG run folder.

Parameters

  • species_dict (dict) – Keys are species labels, values are RMG Species containing the reason for calculating them.

  • unconverged_species (list) – Entries are RMG Species objects for which thermo calculations did not converge in ARC throughout the iterations.
