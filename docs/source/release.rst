Release notes
=============

T3 0.1.0
^^^^^^^^

This is the first version of T3.

T3 is a tool for automated kinetic model generation and refinement.
It has reasonable capabilities to automatically generate kinetic models using RMG-Py_
and to calculate thermodynamic properties and reaction rates for model refinement using ARC_.
