# How to cite

Text form:

    C. Pieters, K.A. Spiekermann, W.H. Green, A. Grinberg Dana,
    The Tandem Tool (T3) for automated chemical kinetic model development,
    version {{T3_VERSION}}, source code: https://github.com/ReactionMechanismGenerator/T3

LaTeX form:

    @misc{T3,
      author = {C. Pieters, K.A. Spiekermann, W.H. Green, A. Grinberg Dana,},
      title = {The Tandem Tool (T3) for automated chemical kinetic model development. https://github.com/ReactionMechanismGenerator/T3, Version {{T3_VERSION}}},
      howpublished = {\url{https://github.com/ReactionMechanismGenerator/T3}},
    }
